# temporal_text_travels

This repository refers to the tool developed during the course "Mineração de dados não-estruturados" ministered by PhD Solange Rezende, PhD Roberta Akemi Sinoara, and PhD Ricardo Marcondes Marcacini at ICMC-USP in 2018, as part of the evaluation process. The proposed tool is employed to visualize relevant terms co-occurrence through time. The employed data was collected from the Websensors Analytics tool (https://www.websensors.net.br/websensors). The results for the case studies presented in the course's final project are under the folder 'case_studies_results'.

Authors: Evaristo Calisto Nhassengo, Saulo Martiello Mastelini, and Tiago Pinho da Silva.
